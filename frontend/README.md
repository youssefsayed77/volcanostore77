# Volcano Frontend Structure

This Vite scaffold serves the frontend from the root `src/` directory.

## Key folders used in this project

- `src/` – React application entry and screens
- `src/components/` – reusable UI components
- `src/assets/` – brand assets such as the Volcano logo
- `src/styles/` – theme and animation styles
- `backend/` – Express and MongoDB API

Run the frontend with `npm run dev`.
